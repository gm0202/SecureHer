## ğŸš€ Getting Started

Follow these steps to set up SecureHer on your local machine.

### Prerequisites

Ensure you have the following installed:

- **Flutter SDK** (latest stable version)
- **Dart SDK** (comes with Flutter)
- **Android Studio** or **Xcode** (for iOS)
- **Git**
- **Firebase account** (for backend setup)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/gm0202/SecureHer.git
   cd SecureHer
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase**
   - Create a new Firebase project
   - Add your Android/iOS app to the project
   - Download and add `google-services.json` (Android) or `GoogleService-Info.plist` (iOS)
   - Enable Firebase Authentication, Realtime Database/Firestore

4. **Run the application**
   ```bash
   flutter run
   ```
